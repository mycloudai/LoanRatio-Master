# Changelog

本项目所有显著变更将记录于此文件。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，
版本号遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [0.2.1] - 2026-04-21

### Fixed

- **全负本金再分配说明缺失**：当所有参还人的原始净本金均为负数时，详情端点的再分配说明现在正确显示"调整后净本金全部归零"，而非输出空的分摊列表
- **空 selectedMonths 静默回退**：预测端点传入 `selectedMonths: []` 空数组时现在返回 400 错误，而非静默回退到 windowMonths 逻辑

### Changed

- **手动模式更新 CP**：手动模式下当月本金按手动比例计入各参还人累计本金 $CP_i$，切回自动模式后利息按 $CP_i/CP$ 比例分摊（而非沿用手动比例）

### Improved

- **历史月份表可折叠列**：默认仅显示净本金和比例两列，累计和利息列可通过"详细列"按钮展开，减少参还人较多时表格过宽问题
- **表格月份列固定**：历史月份表和汇总历史表横向滚动时，月份列始终可见（sticky column）

## [0.2.0] - 2026-04-21

### Added

- **Toast 通知系统**：替换所有 `alert()` 弹窗为 Bootstrap Toast 通知，支持 success/error/info 三种类型
- **按钮加载状态**：所有操作按钮点击后显示 spinner 并禁用，防止重复提交
- **汇总比例进度条**：汇总页各参还人权益比例旁新增可视化进度条
- **标签页计数徽章**：参还人、贷款、月份标签页显示对应数量徽章
- **表单输入标签**：参还人和贷款新增表单增加字段标签说明
- **月份字段列头**：月份录入表单增加利息/本金/还款额列标题
- **默认月份填充**：首次录入月份时自动填充当前年月

### Changed

- **货币千分位格式**：所有金额显示增加千分位分隔符（如 `1,800.00`）
- **删除按钮图标化**：删除、详情、修改按钮增加 Bootstrap Icons 图标
- **ID 列弱化显示**：参还人和贷款表格中 ID 列使用灰色小号字体
- **快照横幅美化**：月度快照横幅采用蓝色渐变背景
- **空状态图标放大**：空状态提示图标更大更显眼
- **快照行提示**：历史月份行增加悬停提示文字

## [0.1.1] - 2026-04-21

### Fixed

- **预测端点浅引用污染**：forecast API 调用后不再污染真实 state 的 `remainingPrincipal`（浅拷贝 → 深拷贝贷款对象）
- **全员欠款场景**：当所有参还人还款不足以覆盖利息时，调整后本金归零，防止累计本金被侵蚀为负数
- **多贷款预测**：forecast 端点按每笔贷款分别计算平均利息/本金，修复多贷款场景预测不准确的问题
- **QA 脚本清理**：退出时清理通过端口号查找的残留服务器进程，防止孤儿进程

### Changed

- **QA 用例 23 更新**：全员欠款场景期望值更新以匹配新的归零逻辑（adj=0，CP 保持首付值）

### Improved

- **UI 视觉升级**：新增 Bootstrap Icons、渐变页头、改进卡片阴影与圆角、更好的表格样式与悬停效果、标签页图标、优化空状态提示

### Added

- **Copilot Instructions**：`.github/copilot-instructions.md` 要求修改代码后运行测试、更新 CHANGELOG 和版本号

## [0.1.0] - 2026-04-21

### Added

- **核心计算引擎**：本金归因模型四步算法（利息分摊 → 原始净本金 → 负本金再分配 → 权益比例更新）
- **首付与零首付支持**：首付按出资比例建立初始权益；零首付首月按 1/n 平摊利息
- **自动 / 手动双模式**：手动模式直接设定比例且不更新累计本金
- **历史月份级联重算**：修改任意历史月份后后续所有月份自动重算
- **参还人管理**：增删改；支持起始月份；删除支持合并 / 直接删除两种策略
- **贷款管理**：增删改；支持原始金额与剩余本金；删除支持合并到其他贷款
- **汇总面板**：展示各参还人累计本金、当前权益、各贷款剩余本金、月度历史表格
- **Excel 导出**：含原生公式，用户可在 Excel 中直接修改数字
- **权益趋势预测**：基于近 N 月平均值预测未来月权益与贷款结清月份
- **版本 / 仓库 / CHANGELOG UI**：页头徽章、仓库链接、内嵌渲染的更新日志模态框
- **本地启动脚本**：`start.sh` / `start.bat` 一键启动
- **JSON 文件存储**：用户自选数据路径，原子写入
- **单元测试套件**：pytest 覆盖 calculator + API，38 用例
- **端到端 QA 脚本**：`qa/run_qa.sh` 覆盖全部核心场景，支持有头 / 无头模式
- **GitHub Actions**：`ci.yml` 单元测试 + lint；`qa.yml` main 推送触发 E2E 测试
- **开源项目规范**：MIT License、README、CONTRIBUTING、CODE_OF_CONDUCT、Issue/PR 模板
