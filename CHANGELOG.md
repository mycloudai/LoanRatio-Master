## [0.1.0] - 2026-04-21

### Added

- **核心计算引擎**：本金归因模型四步算法（利息分摊 → 原始净本金 → 负本金再分配 → 权益比例更新）
- **首付与零首付支持**：首付按出资比例建立初始权益；零首付首月按 1/n 平摊利息
- **自动 / 手动双模式**：手动模式直接设定比例且不更新累计本金
- **历史月份级联重算**：修改任意历史月份后后续所有月份自动重算
- **参还人管理**：增删改；支持起始月份；删除支持合并 / 直接删除两种策略
- **贷款管理**：增删改；支持原始金额与剩余本金；删除支持合并到其他贷款
- **汇总面板**：展示各参还人累计本金、当前权益、各贷款剩余本金、月度历史表格
- **Excel 导出**：含原生公式，用户可在 Excel 中直接修改数字
- **权益趋势预测**：基于近 N 月平均值预测未来月权益与贷款结清月份
- **版本 / 仓库 / CHANGELOG UI**：页头徽章、仓库链接、内嵌渲染的更新日志模态框
- **本地启动脚本**：`start.sh` / `start.bat` 一键启动
- **JSON 文件存储**：用户自选数据路径，原子写入
- **单元测试套件**：pytest 覆盖 calculator + API，38 用例
- **端到端 QA 脚本**：`qa/run_qa.sh` 覆盖全部核心场景，支持有头 / 无头模式
- **GitHub Actions**：`ci.yml` 单元测试 + lint；`qa.yml` main 推送触发 E2E 测试
- **开源项目规范**：MIT License、README、CONTRIBUTING、CODE_OF_CONDUCT、Issue/PR 模板
